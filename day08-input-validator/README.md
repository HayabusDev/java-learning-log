# Day08 - Input Validation Practice

## 作成物
ユーザー情報入力コンソールアプリ（名前・年齢・居住地確認）

## 機能
- 空文字チェック付き文字列入力
- Yes / No 判定入力
- 範囲指定付き整数入力
- 入力バリデーションによる再入力制御

## 学習内容
- Scanner + nextLine による安全な入力処理
- trim / Normalizer による入力正規化
- try-catch を用いた数値変換処理
- 再利用可能な入力メソッド設計

## 気づき・工夫
- 入力処理を Controller に集約することで、呼び出し側を簡潔にできた
- boolean で Yes/No を扱うことで分岐を安全にした
- 入力エラー時に必ず復帰できる構造を意識した

## 将来へのつながり
- CRUDアプリや業務系アプリのフォーム入力処理に応用できる
- InputUtil クラスとして共通化可能
