# Day11 - Word Book App（単語帳アプリ）

## 作成物
コンソール型 英単語帳アプリ

英単語と意味を登録し、検索・一覧・削除ができる簡易辞書アプリです。

---

## 機能一覧
- 英単語の登録
- 英単語の一覧表示（アルファベット順）
- キーワード検索
- 英単語の削除
- 登録・削除時の確認（y/n）

---

## 使用技術・構成
- Java（コンソールアプリ）
- HashMap によるデータ管理
- MVC風構成（Controller / Service / Repository）
- Result + ErrorCode によるエラーハンドリング
- Comparator によるソート処理

---

## 学習内容
- Map → List 変換処理
- for文による検索処理
- Comparator を用いた並び替え
- ジェネリクス（Result<T> / SafeAction<T>）
- enum を使ったエラー管理
- 例外処理の共通化設計
- 入力チェックの共通化

---

## 工夫点
- boolean ではなく Result クラスで処理結果を管理
- ErrorCode によるエラー分類の統一
- Service 層で業務ロジックを集約
- executeSafely による例外の一括管理
- ソート時にリストをコピーし、副作用を防止

---

## 発見・反省
- エラーハンドリング設計に想定以上に時間を使った
- 設計を考えることで、コードの見通しが良くなった
- 小規模アプリでも責務分離の重要性を実感した
